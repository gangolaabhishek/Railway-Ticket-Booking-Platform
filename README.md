# Railway Ticket Booking Platform

## Overview

Enterprise-grade Railway Ticket Booking Platform built using Java Spring Boot Microservices.

## Tech Stack

- Java
- Spring Boot
- Spring Cloud
- MySQL
- Hibernate
- JPA
- Kafka
- Redis
- Docker
- Kubernetes
- Jenkins
- OpenFeign
- WebClient
- API Gateway
- Eureka
- Config Server

## Modules

- User Service
- Train Service
- Schedule Service
- Seat Service
- Booking Service
- Payment Service
- PNR Service
- Notification Service
- Audit Service
- Admin Service

## Features

- Train Search
- Ticket Booking
- PNR Status
- Dynamic Fare
- Waitlist/RAC
- Refunds
- QR Code Ticket
- Notifications
- Booking History
- Distributed Tracing
- Saga Pattern
- Kafka Events
- JWT Security (planned)
